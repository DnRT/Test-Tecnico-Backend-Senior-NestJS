# Test-T-cnico-Backend-Senior-NestJS
Test para medir las competencias, teóricas y practicas, para postular al puesto de Técnico Backend Senior en Finmarkets

# Ejecucion del proyecto API

Para comenzar debe estar en la carpeta del proyecto con un 
``` cd taskmanager-api ```

Para luego realizar un
``` npm run start ```

Y mediante el navegador o software especializados, como Postman, usando como raiz
``` localhost:3000 ```
